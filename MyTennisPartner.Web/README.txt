Build notes:
--------------
To prepare your environment to develop and run this project:

To add sass loader to project:
>npm install --save sass-loader node-sass

ANY changes to package.json should be followed by an 'npm install' command at the dos prompt.

Last successful build was with NPM version 6.4.1, webpack 4.24.0

Build batch file webpackbundle.bat can be used to run npm i, webpack, shrink, etc.
